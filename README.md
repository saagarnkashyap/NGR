## Features

- **Responsive** across all devices
- Smooth **scroll animations** with GSAP
- Modular components for easy updates
- Downloadable resume (PDF)
- Cleanly structured sections:
  - Hero + About
  - Experience Timeline
  - Education
  - Projects Grid
  - Contact Info + Links
